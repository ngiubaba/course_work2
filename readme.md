# Поиск и фильтрация вакансий с HeadHunter API

## Создан для отработки навыков при обучении на backend-разработчика ##

*Данный проект будет дополняться и обновляться*

### Функционал:
 * Получение вакансий через API hh.ru
 * Фильтрация вакансий по ключевым словам в описании
 * Фильтрация вакансий по диапазону зарплат
 * Сортировка и вывод топ-N вакансий
 * Сохранение и чтение вакансий из JSON-файла
 * Исключение дублирующихся вакансий при сохранении
 * Удаление вакансий по ссылке
 * Валидация входных данных (название, ссылка, зарплата)
 * Работа с абстрактными базовыми классами
 * Модульное тестирование всех ключевых компонентов

### Для запуска:
*PyCharm или любая среда с поддержкой Python*

1. Клонируйте репозиторий:
```
git clone https://github.com/ngiubaba/coursepaper1.git
cd course-work2
```

2. Активируйте виртуальное окружение poetry:
```
poetry shell
```

3. Установите зависимости по команде:
```
poetry install
```

4. Запустите файл **main.py**:
```
python main.py
```

## Тестирование
### Добавлены модули для теста:
#### Модуль `test_api_check.py`:
* Тест метода подключения к API
* Тест метода получения вакансий

#### Модуль `test_file_module.py`:
* Тест сохранения и загрузки из файла
* Тест удаления дубликатов

#### Модуль `test_user_iteration.py`:
* Тест фильтрации по ключевым словам
* Тест фильтрации по зарплате
* Тест вывода топ вакансий

#### Модуль `test_vacancy_module.py`:
* Тест создания объекта вакансии
* Тест преобразования из словаря
* Тест преобразования в словарь

### Для запуска тестов:
1. Установите зависимости:
```
poetry add --group dev pytest pytest-mock
```

2. Запустите тесты командой:
```pytest```

3. Для покрытия тестов:
```
poetry add --group dev pytest-cov
pytest --cov=src --cov-report=html
```

*Отчет будет доступен в папке `htmlcov/index.html`*
